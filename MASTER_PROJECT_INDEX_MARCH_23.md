## 📞 SUPPORT & TROUBLESHOOTING

### Common Issues & Solutions

#### "Port 3506 already in use"
```powershell
# Kill existing Node process
taskkill /IM node.exe /F
# Restart
npm run dev
```

#### "Database connection error"
```
1. Check connection string in .env
2. Verify database server is running
3. Check credentials
4. Check firewall rules
```

#### "Export not generating 16 files"
```
1. Check console errors (npm run dev)
2. Check database query logs
3. Verify data exists in tables
4. Check disk space for ZIP
```

#### "Drug data showing false/mock data"
```
1. Restart backend: npm run dev
2. Check database has no records for VN
3. Clear browser cache
4. Check network tab in DevTools
```
