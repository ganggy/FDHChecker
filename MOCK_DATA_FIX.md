# 🔧 Mock Data Fix - ข้อมูลแตกต่างตาม VN

## 📋 **ปัญหาที่พบ**
- ข้อมูลยาและค่าบริการเหมือนกันทุก VN
- ผู้ใช้ไม่สามารถแยกแยะข้อมูลแต่ละผู้ป่วยได้
- ทำให้การทดสอบไม่สมจริง

## ✅ **วิธีแก้ไข**

### **1. เพิ่มข้อมูล Mock สำหรับ VN ต่างๆ**

#### **ข้อมูลยา (Prescriptions)**
```javascript
VN: 690351541353 → Aspirin 80mg Tab (1 รายการ)
VN: 690351555432 → Aspirin + Cephalexin (2 รายการ)  
VN: 690350441149 → Paracetamol + Amoxicillin + Omeprazole (3 รายการ)
VN: 2601234      → Paracetamol + Amoxicillin (2 รายการ)
```

#### **ข้อมูลค่าบริการ ADP**
```javascript
VN: 690351541353 → ค่าบริการ OPD (1 รายการ)
VN: 690351555432 → ค่าบริการ OPD + ค่าจ่ายยา (2 รายการ)
VN: 690350441149 → ค่าแพทย์ + ค่าแลป + ค่าเอกซเรย์ (3 รายการ)
VN: 2601234      → ค่าบริการผู้ป่วยใน (1 รายการ)
```

### **2. การทดสอบ API**

```bash

curl "http://localhost:3001/api/hosxp/services/690351555432"  
curl "http://localhost:3001/api/hosxp/services/690350441149"
```

## 🧪 **ผลการทดสอบ**

### **VN: 690351541353**
- **ยา**: Aspirin 80mg Tab (30 เม็ด) = 15฿
- **ค่าบริการ**: ค่าบริการ OPD (ADP001) = 300฿ ✅ เบิกได้

### **VN: 690351555432**
- **ยา**: Aspirin (15 เม็ด) + Cephalexin 500mg (14 แคปซูล) = 77.5฿
- **ค่าบริการ**: 
  - ค่าบริการ OPD (ADP003) = 200฿ ✅ เบิกได้
  - ค่าจ่ายยา = 0฿ ❌ ไม่เบิก

### **VN: 690350441149**  
- **ยา**: Paracetamol + Amoxicillin + Omeprazole = 179.6฿
- **ค่าบริการ**:
  - ค่าบริการแพทย์ (ADP001) = 500฿ ✅ เบิกได้
  - ค่าตรวจแลป (ADP002) = 200฿ ✅ เบิกได้
  - ค่าเอกซเรย์ = 0฿ ❌ ไม่เบิก
