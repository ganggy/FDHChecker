## ต้องแยกช่องทางเบิกก่อนตั้ง rule

| ช่องทาง | ใช้กับ |
|---|---|
| FDH/e-Claim | รายการ claim/export FDH หรือ e-Claim เช่น OP/IP, ADP/FS บางกลุ่ม |
| 43 แฟ้ม | รายการที่เน้นความครบของข้อมูลมาตรฐาน เช่น พอกเข่า |
| MOPH Claim | DMHT Lab และ Vaccine EPI/dT/HPV/aP ที่ส่ง API MOPH Claim |
| KTB | PPFS หลายรายการ เช่น Hep B/C, syphilis, anemia, TDAS, osteoporosis |
| NTIP/TB Data Hub | Latent TB/IGRA |
| อื่นๆ | ระบบเฉพาะ เช่น VMI, renal, approval พิเศษ |

## รายการที่ควรผูก rule ใน FDHChecker

| Rule | เงื่อนไขตรวจจับจากข้อมูล | จุดเสี่ยง |
|---|---|---|
| ซิฟิลิสประชาชนทั่วไป/เพศชาย | เพศชาย, lab Treponema/Syphilis/TPHA/TPPA/RPR/VDRL | ต้องแยกจาก ANC syphilis และตรวจจำนวนครั้งตามกลุ่มเป้าหมาย |
| Hepatitis C | เกิดก่อน พ.ศ. 2535, ICD `Z11.5`, lab Anti-HCV | ตั้งแต่ 1 มิ.ย. 2568 บันทึกผ่าน KTB |
| Hepatitis B | เกิดก่อน พ.ศ. 2535, ICD `Z11.5`, lab HBsAg | เงื่อนไขเดียวกับ Hep C แต่ lab เป็น HBsAg |
| พอกเข่า | อายุ 40 ปีขึ้นไป และมี Dx `M17` หรือ `U57.53`; ดึงมาแม้ยังขาดหัตถการ | เป็นกองทุนที่เบิกผ่าน 43 แฟ้ม ไม่ใช่กองทุนพิเศษ |
| โลหิตจางเด็ก | อายุ 6-12 เดือน หรือ 3-6 ปี และมี Hb/Hct | ต้องไม่ปะปนกับ iron supplement |
| Latent TB | มี IGRA/ELISA/Chemiluminescence/Fluorescence | ต้องแยกระหว่างเจาะเลือด/ส่ง specimen/lab |

## หมายเหตุสำหรับการพัฒนา

- ถ้าทำเป็นหน้ากองทุนพิเศษ ให้แยกเฉพาะรายการที่เป็น FS/PPFS จริง เช่น ซิฟิลิส, Hep B/C, โลหิตจาง, Latent TB
- พอกเข่าควรอยู่ในกลุ่มตรวจ 43 แฟ้ม/บริการแพทย์แผนไทย ไม่ควรใส่เป็นกองทุนพิเศษ
- กรณี Hep B/Hep C ใช้ Dx `Z11.5` ไม่ใช่ `115`
- ข้อมูล text ดิบมีไว้สำหรับค้นคำ/ตรวจย้อนแหล่งอ้างอิง ส่วนไฟล์นี้ใช้เป็น working note สำหรับ mapping rule
