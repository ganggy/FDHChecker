## 🚀 **Test Case 1: ข้อมูลยาแตกต่างตาม VN**

### **Expected Result:**
- VN `690350441149` → 3 รายการยา
- VN `2601234` → 2 รายการยา
- VN อื่นๆ → 1 รายการยา

### **Steps:**
1. เปิด http://localhost:5175/staff
2. คลิกแถว VN `690350441149` → ควรเห็น **3 รายการยา**:
   - Paracetamol 500mg Tab (20 เม็ด)
   - Amoxicillin 250mg Cap (21 แคปซูล)
   - Omeprazole 20mg Cap (10 แคปซูล)
3. คลิกแถว VN `2601234` → ควรเห็น **2 รายการยา**:
   - Paracetamol 500mg Tab (20 เม็ด)
   - Amoxicillin 250mg Cap (21 แคปซูล)

### ✅ **Pass Criteria:**
- [x] ข้อมูลยาแตกต่างกันตาม VN
- [x] จำนวนรายการถูกต้อง
- [x] ไม่มี ADP Code ในส่วนยา (เพราะยาไม่ได้เบิกผ่าน ADP)

## 🏥 **Test Case 2: ข้อมูลค่าบริการ ADP Code**

### **Expected Result:**
- มีส่วน "ข้อมูลค่าบริการ ADP Code" แยกต่างหาก
- แสดง Highlight เขียว/แดง ตามสถานะการเบิก
- มีสถิติการเบิก

### **Steps:**
1. คลิกแถว VN `690350441149`
2. ดูส่วน **"ข้อมูลค่าบริการ ADP Code"** ด้านบน
3. ควรเห็น **3 รายการบริการ**:
   - 🟢 **ค่าบริการแพทย์** (ADP001) - เบิกได้ 500฿
   - 🟢 **ค่าตรวจวิเคราะห์** (ADP002) - เบิกได้ 200฿  
   - 🔴 **ค่าเอกซเรย์** - ไม่มี ADP - ไม่เบิก
4. ดูสถิติ: **เบิกได้: 2/3**
5. ยอดรวมค่าบริการที่เบิกได้: **700฿**

### ✅ **Pass Criteria:**
- [x] แสดงตารางค่าบริการแยกจากยา
- [x] Highlight สีเขียวสำหรับรายการที่เบิกได้
- [x] Highlight สีแดงสำหรับรายการที่ไม่เบิก
- [x] แสดง ADP Code อย่างชัดเจน
- [x] สถิติการเบิกถูกต้อง

## 📋 **Test Case 3: ส่วนข้อมูลยา (ปรับปรุงแล้ว)**

### **Expected Result:**
- ไม่มี Highlight เพราะยาไม่เบิกผ่าน ADP
- มีหมายเหตุอธิบาย
- แสดงเฉพาะข้อมูลพื้นฐานของยา

### **Steps:**
1. ดูส่วน **"ข้อมูลยา"** ด้านล่าง
2. ตารางควรแสดง:
   - รหัสยา
   - ชื่อยา
   - จำนวน
   - ราคาต่อหน่วย
   - ราคารวม
3. **ไม่มีคอลัมน์ ADP Code** (เพราะยาไม่เกี่ยวข้อง)
4. ดูหมายเหตุ: "ยาไม่มีรหัส ADP Code เพราะเป็นรายการยา ADP Code ใช้สำหรับค่าบริการเท่านั้น"

### ✅ **Pass Criteria:**
- [x] ไม่มี Highlight ในตารางยา
- [x] ไม่มีคอลัมน์ ADP Code
- [x] มีหมายเหตุอธิบายความแตกต่าง
- [x] แสดงราคารวมค่ายาถูกต้อง
